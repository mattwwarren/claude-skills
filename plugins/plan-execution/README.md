# plan-execution

Phase-by-phase execution of approved markdown plans, dispatching parallel sub-agents for independent tasks.

## Skills

- **plan-executor** — Take an approved plan (markdown with H2 phase headers and checkbox tasks) and execute it phase-by-phase. Spawns parallel sub-agents where possible. Ships templates under `skills/plan-executor/templates/`.

## Install

```text
/plugin install plan-execution@claude-skills
```
